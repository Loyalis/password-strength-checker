🔍 Project Breakdown & Code Highlights

🧪 Key Features:
  • Checks password strength using character types, length, and patterns
  • Calculates entropy to estimate password randomness
  • Flags weak or leaked passwords using rockyou.txt
  • Hashes and stores strong passwords in a local SQLite database
  • Displays real-time feedback using Flask and Bootstrap
🧠 Important Code Concepts:

App Setup & Secrets: I use Flask to create routes and render pages. The app’s secret key is stored in a hidden .env file using dotenv so it’s never exposed directly in the code.

Loading Leaked Passwords: I included a function that loads leaked passwords from the famous rockyou.txt list into a Python set. This lets me instantly check if a user’s password has been exposed in real-world breaches.

Checking Password Strength: I use Python’s re module (regular expressions) to check if a password has lowercase, uppercase, digits, symbols, and a minimum length of 8 characters. It also scans for names like "gabriel" to discourage predictable personal patterns.

Entropy Calculation: This formula estimates how hard a password is to guess:
entropy = length * log2(character_set_size) The more varied the characters and longer the password, the better.

Hashing & Storage: If a password is marked “Strong,” it gets hashed using Werkzeug’s generate_password_hash() before being saved. This simulates how real apps never store plain passwords—only hashes.

Database & Initialization: On app startup, a function checks if the passwords table exists in SQLite. If not, it creates it, and from there I can store and retrieve strong password hashes for display.

Routing & Templates: The / route handles both GET and POST requests, displaying the form and accepting input. The /about route loads this page. Flask handles the logic, while Bootstrap makes it look good.

Overall, this project helped me connect real-world security concepts with practical code. I’m still learning every day, but I now have a clearer understanding of what it takes to build safer, smarter tools—and the confidence that I can keep growing from here.
